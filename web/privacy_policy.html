Política de Privacidad

Última actualización:

1. Información General

La aplicación "Horas Médicas" es una herramienta diseñada para facilitar el acceso rápido a las páginas web de clínicas médicas en Rancagua, permitiendo a los usuarios reservar horas médicas de manera sencilla. Esta política de privacidad describe cómo manejamos tus datos personales.

2. Datos que Recopilamos

La aplicación "Horas Médicas" NO recopila, almacena ni transmite ningún dato personal.

La aplicación funciona como un simple navegador de enlaces que abre las páginas web de las clínicas médicas en tu navegador predeterminado. No tenemos acceso a:

  • Información personal que ingreses en los sitios web de las clínicas
  • Datos de navegación o historial
  • Información de tu dispositivo
  • Ubicación o datos de geolocalización
  • Contactos, fotos u otros archivos

3. Almacenamiento de Datos

No almacenamos ningún dato. La aplicación no guarda información localmente ni en servidores externos. Toda la funcionalidad se realiza a través de tu navegador web cuando accedes a los sitios de las clínicas.

4. Uso de los Datos

Como no recopilamos datos, no hay uso de información personal. La aplicación simplemente:

  • Muestra una lista de clínicas médicas disponibles
  • Abre el sitio web de la clínica seleccionada en tu navegador

Cualquier información que proporciones directamente a las clínicas médicas a través de sus sitios web está sujeta a las políticas de privacidad de cada clínica respectiva.

5. Compartir Información

No compartimos, vendemos ni alquilamos datos personales porque no recopilamos ninguno.

6. Permisos de la Aplicación

La aplicación requiere los siguientes permisos mínimos:

  • Acceso a Internet: Necesario para abrir los sitios web de las clínicas en tu navegador

La aplicación NO requiere ni solicita permisos para:

  • Ubicación o GPS
  • Cámara o fotos
  • Micrófono
  • Contactos
  • Almacenamiento de archivos
  • Notificaciones

7. Enlaces a Sitios de Terceros

La aplicación contiene enlaces a sitios web de clínicas médicas que son operados por terceros. No tenemos control sobre el contenido, políticas de privacidad o prácticas de estos sitios web. Te recomendamos revisar las políticas de privacidad de cada clínica antes de proporcionar cualquier información personal.

8. Seguridad

Como no recopilamos ni almacenamos datos, no hay riesgo de exposición de información personal a través de nuestra aplicación. Sin embargo, cuando accedas a los sitios web de las clínicas, asegúrate de que estés usando una conexión segura (HTTPS) y de que tu dispositivo esté protegido.

9. Cambios en esta Política

Podemos actualizar esta política de privacidad ocasionalmente. Te notificaremos de cualquier cambio significativo mediante una actualización de la aplicación. La fecha de la última actualización se muestra al inicio de este documento.
